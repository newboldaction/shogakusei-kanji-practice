# 漢字データ

## 収録内容

`kyoiku-kanji.json` は **常用漢字2140字**（教育漢字1026字＋中学以降で習う常用漢字1114字）のデータを収録しています。

| 学年 | 字数 |
|---|---|
| 小1 | 80 |
| 小2 | 160 |
| 小3 | 200 |
| 小4 | 202 |
| 小5 | 193 |
| 小6 | 191 |
| 中学以降 (`grade: null`) | 1114 |
| 合計 | 2140 |

`grade` が `1〜6` のものは小学校で習う教育漢字、`null` は中学校以降で習う常用漢字です。
学年タブには小学校分のみ表示し、中学以降の漢字は「直接入力欄」に書いた場合のみ書き順マスに使われます。

## データ構造

```json
{
  "一": { "grade": 1, "on": ["イチ", "イツ"], "kun": ["ひと-", "ひと.つ"], "strokes": 1 }
}
```

## 備考

- kanjiapi.dev の `grade` 値は2020年改訂前の旧配当に基づいているため、本プロジェクトでは学年情報のみ文科省2020年版配当表で上書きしている（都道府県名由来の20字を含む）。
- 常用漢字2136字を超える4字（「璽」など旧字体）が `kanjiapi.dev /v1/kanji/joyo` には含まれており、収録総数は **2140** となっている。
- 常用漢字外（人名漢字や難読漢字）は本データに含まれず、書き順SVGも未収録のため、お手本マスは「書き順未収録」のフォールバック表示になる。
